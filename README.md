# 🪙 BTC DCA Portfolio - เว็บบันทึกการออมบิตคอยน์พรีเมียม

> **เว็บบันทึกข้อมูลการออมแบบ DCA (Dollar-Cost Averaging) ในบิตคอยน์ส่วนตัวของคุณ บันทึกราคาทุน คำนวณ Satoshi และติดตามเป้าหมายสู่ 1 BTC แบบเรียลไทม์ ด้วยดีไซน์ดำส้มสุดหรูหราแบบพรีเมียม**

🚀 **ทดลองใช้งานจริงได้ที่นี่**: [https://dca-btc-webapp.vercel.app](https://dca-btc-webapp.vercel.app)

---

## ✨ คุณสมบัติเด่น (Features)
* **🎨 Premium Glassmorphism UI**: การออกแบบระดับพรีเมียม ธีมสีดำส้มสว่างเรืองแสง พร้อมสลับธีมระหว่าง **มืดหรูหรา (Dark Mode)** และ **ดำสนิท (Pitch Black)**
* **📊 สถิติออมบิตคอยน์ครบวงจร**: แสดงผลมูลค่าพอร์ตปัจจุบัน, ต้นทุนทั้งหมด, กำไร/ขาดทุนแบบเรียลไทม์ และค่าเฉลี่ยต้นทุนแบบประเมินค่าอย่างละเอียด
* **💡 DCA Guide & FAQ**: มีคู่มือแนะนำขั้นตอนการออม กลยุทธ์ DCA และคำแนะนำความปลอดภัยซ่อนอยู่ในตัวเว็บได้อย่างแนบเนียน
* **📈 Interactive Chart**: กราฟแนวโน้มพอร์ตสะสม (ApexCharts) แสดงการเปรียบเทียบมูลค่าพอร์ตกับเงินทุนแบบมีชีวิตชีวาและสอดคล้องกับธีมมืด
* **📱 Share Card 3D**: ดาวน์โหลดภาพการ์ดสรุปพอร์ตโฟลิโอส่วนตัว พร้อมสัญลักษณ์ 3D Bitcoin Path แท้ เพื่อแชร์ลงโซเชียลมีเดียได้อย่างลงตัว
* **💾 Data Import/Export (CSV/JSON)**: ระบบนำเข้าและสำรองประวัติการออมของคุณในรูปแบบไฟล์เพื่อกันข้อมูลสูญหาย

---

## 🔒 ปลอดภัยสูงสุด ความเป็นส่วนตัว 100% (Privacy First)
* **No Servers**: ข้อมูลประวัติธุรกรรมของคุณทั้งหมดจะถูกบันทึกไว้ใน `LocalStorage` ของเบราว์เซอร์บนเครื่องคุณเท่านั้น
* **ไม่มีการส่งข้อมูลการเงินใด ๆ ออกไปยังฐานข้อมูลภายนอก** ทำให้ปลอดภัยจากการตรวจสอบ การรั่วไหล หรือการถูกแฮ็กระบบฐานข้อมูลส่วนกลาง
* **Privacy Mode**: ปุ่มไอคอนรูปดวงตาช่วยซ่อนยอดเงินรวมทั้งหมดขณะเปิดใช้งานในที่สาธารณะ

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* **Core**: HTML5, Vanilla JavaScript (ESM Modules)
* **Styling**: Vanilla CSS (Custom Design System, Glassmorphic effects)
* **Libraries**: ApexCharts (สำหรับการวาดกราฟความเติบโต)
* **Build Tool**: Vite (รองรับการคอมไพล์งานที่รวดเร็วและเบา)

---

## 💻 วิธีการรันบนเครื่องตนเอง (Local Installation)

หากต้องการนำโค้ดไปเปิดใช้งานหรือศึกษาในเครื่องส่วนตัว:

1. โคลนคลังโค้ดนี้ลงเครื่องคอมพิวเตอร์ของคุณ
2. ติดตั้ง Dependencies:
   ```bash
   npm install
   ```
3. รันพัฒนาเครื่องในระบบจำลอง (Local Dev Server):
   ```bash
   npm run dev
   ```
4. เข้าหน้าเว็บทดสอบที่ลิงก์: `http://localhost:5173`
